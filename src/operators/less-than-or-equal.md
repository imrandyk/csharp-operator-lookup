The `<=` returns true if the left hand operand is less than or equal to the right hand.

```cs
Console.WriteLine(7 <= 5); // False
Console.WriteLine(5 <= 5); // False
Console.WriteLine(0 <= 5); // True
Console.WriteLine(5 <= 5); // True
```
